Non, l'acide palmitoléique n'est pas uniquement d'origine végétale. Voici les principales sources et origines de l'acide palmitoléique :

1. Sources végétales :
- L'huile de noix de macadamia (Macadamia integrifolia) contient environ 17% d'acide palmitoléique[1][5].
- L'huile d'argousier (Hippophae rhamnoides) contient 19-29% d'acide palmitoléique[1][5].
- L'huile d'avocat contient environ 7,3% d'acide palmitoléique[2].
- On en trouve également des traces dans l'huile de palme et de colza[2].

2. Sources animales :
- La matière grasse lactique (produits laitiers) contient environ 2,6% d'acide palmitoléique[2].
- L'huile de vison, d'origine animale, contient environ 17% d'acide palmitoléique[7].

3. Source endogène (production par l'organisme) :
- La quasi-totalité de l'acide palmitoléique dans le corps humain est d'origine endogène[2].
- Il est biosynthétisé à partir de l'acide palmitique par l'action de l'enzyme Stearoyl-CoA désaturase-1[1].
- Cette désaturation est insulinodépendante[2].

4. Présence dans le corps humain :
- L'acide palmitoléique est un constituant commun des glycérides du tissu adipeux humain[1][5].
- Il est présent dans tous les tissus, mais généralement à des concentrations plus élevées dans le foie[1][5].

En conclusion, bien que certaines sources végétales soient riches en acide palmitoléique, il n'est pas exclusivement d'origine végétale. Il est également présent dans certaines sources animales et est largement produit par le corps humain lui-même.
